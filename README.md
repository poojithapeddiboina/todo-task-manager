# Todo Task Manager

## Description
A simple Todo Task Manager application that allows users to add, complete, and delete daily tasks.
This project was built as part of Skill Craft internship practice.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
